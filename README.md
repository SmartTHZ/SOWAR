# نظام سوار - حماية ضحايا العنف الأسري

**نظام سوار** هو مشروع يهدف إلى حماية ضحايا العنف الأسري من خلال متابعة الحالات الطارئة للأساور الذكية وعرضها على خريطة تفاعلية.

---

## الفكرة

يوفر المشروع متابعة لحظية للحالات الطارئة عبر الأساور الذكية، ويصنف الحالات حسب شدة الطوارئ لتسهيل اتخاذ الإجراءات السريعة من فرق الطوارئ.

---

## المميزات

- عرض الحالات على خريطة تفاعلية باستخدام **Leaflet.js**.  
- تمييز الحالات حسب شدة الطوارئ: حرجة، عالي، متوسط.  
- عرض تفاصيل كل حالة:  
  - السوار  
  - الموقع  
  - الحالة (تم نزع السوار، استغاثة، إلخ)  
  - نبض القلب  
  - عدد الضغطات على السوار  
- إشعارات فورية عند حدوث أي حالة حرجة.  
- تجربة تفاعلية مباشرة بدون الحاجة لقاعدة بيانات (تحديثات الوقت والتقارير محاكاة).

---

## طريقة التشغيل

1. تأكد من وجود جميع الملفات:  
   - `index.html`  
   - مجلد `_sdk/` يحتوي على SDK المستخدمة  

2. افتح ملف `index.html` في متصفح حديث (مثل Chrome أو Edge).  

3. سيتم عرض الخريطة والحالات مباشرة مع إمكانية تجربة الوظائف التفاعلية.

---

## التقنيات المستخدمة

- HTML / CSS / JavaScript  
- [Tailwind CSS](https://tailwindcss.com/)  
- [Leaflet.js](https://leafletjs.com/) للخرائط  
- SDK داخلي موجود في `_sdk/`

---

- يمكن ربطه بقاعدة بيانات لاحقًا لتحديث الوقت والحالات بشكل حقيقي.  
- جميع البيانات الحالية محاكاة لأغراض العرض فقط.
